This project is a frontend clone of the Nike website landing page created using HTML and CSS. The goal of this project was to practice building modern website layouts and improve frontend development skills.

The website replicates the design style of the global sports brand Nike, which is known for designing and selling athletic footwear, apparel, and sports equipment worldwide.

This project focuses on creating a clean UI, responsive layout, and visually appealing product sections similar to a real e-commerce landing page.
